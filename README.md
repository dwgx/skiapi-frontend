# SKIAPI Frontend

SKIAPI 的 Admin 控制台前端，对接 [NewAPI](https://github.com/Calcium-Ion/new-api)。

基于 **React 19 + Vite + MUI v7**。

---

## 技术栈

- React 19
- Vite
- MUI (Material UI) v7
- ESLint

## 开发

```bash
git clone https://github.com/dwgx/skiapi-frontend.git
cd skiapi-frontend
npm install
npm run dev
```

## 构建

```bash
npm run build
```

预览构建产物：

```bash
npm run preview
```
